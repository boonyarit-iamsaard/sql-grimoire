# Triage Labels

The skills use five canonical triage roles. This file maps those roles to the label strings used by
this repository's issue tracker.

| Label in mattpocock/skills | Label in our tracker | Meaning                                   |
| -------------------------- | -------------------- | ----------------------------------------- |
| `needs-triage`             | `needs-triage`       | Maintainer needs to evaluate this issue   |
| `needs-info`               | `needs-info`         | Waiting on reporter for more information  |
| `ready-for-agent`          | `ready-for-agent`    | Fully specified for unattended agent work |
| `ready-for-human`          | `ready-for-human`    | Requires human implementation             |
| `wontfix`                  | `wontfix`            | Will not be addressed                     |

When a skill mentions a role, such as "apply the unattended-agent triage label," use the
corresponding label string from this table.

Edit the right-hand column to match the vocabulary used by the configured issue tracker.
